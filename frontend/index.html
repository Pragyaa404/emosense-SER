About the Project

What is Speech
Emotion Recognition?

Speech Emotion Recognition (SER) is the task of automatically identifying human emotional states from audio signals. EmoSense combines multiple deep learning architectures trained on a merged, balanced dataset.

The system processes raw audio through two parallel feature extraction pipelines — Mel spectrograms and MFCCs — feeding into an ensemble of 5 specialized models whose predictions are combined via weighted soft voting.

  • ✓ Balanced dataset merged from two real-world sources
  • ✓ Dual feature pipeline — Mel spectrograms + MFCC
  • ✓ CBAM attention — channel + spatial attention
  • ✓ 5-fold cross-validation for robust evaluation
  • ✓ Continuous learning from real user feedback

How EmoSense Works

Two parallel feature pipelines feed into a 5-model ensemble for maximum accuracy

🎛️
Pipeline A — Mel
Feeds CNN-CBAM ensemble
1 Raw WAV audio (16kHz)
↓
2 Mel Spectrogram (128×128)
↓
3 Trim silence + normalise
↓
4 CNN + CBAM Attention ×3
↓
5 Soft-vote ensemble
🌊
Pipeline B — MFCC
Feeds BiLSTM + EmoFormer
1 Raw WAV audio (16kHz)
↓
2 MFCC Features (128×40)
↓
3 Pad / truncate to 128 frames
↓
4 CNN → BiLSTM + Attention
CNN → Transformer Encoder
↓
5 Average predictions
Weighted Soft Voting Ensemble

Final prediction = average of all 5 models' probability outputs

Continuous Learning

Every prediction's audio is saved. When you confirm or correct a result, that label is stored for periodic retraining — improving the model over time using real user data.
